Feature update

BSP-Doc in BETA

The BSP level documentation is still incomplete and considered in BETA quality.

Currently the BSP-Doc covers LAN9662, LAN9668 (using EVB: UNG8290 and UNG8291), Sparx5 (using EVB PCB134 and PCB135) and peripheral devices found LAN966X boards like: I2C, MCAN, MMC, OTP, etc..

P-NET and MERA

Profinet and MERA(Microchip Ethernet RTE API) packages were added in buildroot. This is possible to run only on UNG8309.

    • MAS_SW-420: 2.5G_Port: When speed is configured to forced, only auto-negotiation is turned off and port is always advertising 2.5G and 1G speeds, also port status is always UP with 2.5G irrespective of the configured forced speed.
